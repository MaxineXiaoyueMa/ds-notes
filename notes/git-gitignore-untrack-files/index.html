Git gitignore untrack files

gitignore and untrack ignored files

  1. create .gitignore file: e.g. echo .DS_Store >> .gitignore ```shell # OSX .DS_Store

# Environment .venv

# Mkdocs /site/ ```

  1. untrack everything that is now in .gitignore:
  2. commit all changes
  3. remove everything from the repo: git rm -r --cached . (recursively remove all files from the index) or, remove a specific file: git rm --cached some.txt
  4. re add everything: git add .
  5. commit: git commit -m '.gitignore fix'
